# Сайт кафе «Вкусненько»

Это простой статический сайт-визитка: `index.html`, `styles.css`, `script.js`.

## Что заменить перед публикацией

1. В блоке «Меню» замените `от ___ ₽` на реальные цены.
2. Добавьте реальные фото блюд:
   - можно заменить блоки `.photo-card` на `<img src="images/photo1.jpg">`;
   - или оставить как есть для первого запуска.
3. Уточните график работы и условия доставки.
4. При наличии ссылок на Telegram/VK/Instagram добавьте кнопки в блок «Контакты».

## Как открыть локально

Просто откройте файл `index.html` в браузере.

## Как загрузить на обычный хостинг

1. Купите домен, например `vkusnenko-feo.ru`.
2. Возьмите простой виртуальный хостинг.
3. Загрузите файлы `index.html`, `styles.css`, `script.js` в папку сайта:
   - часто это `public_html`, `www` или папка домена.
4. Включите SSL/HTTPS в панели хостинга.
5. Проверьте сайт с телефона.

## Как загрузить на Netlify

1. Зарегистрируйтесь на Netlify.
2. Нажмите Add new site → Deploy manually.
3. Перетащите папку `vkusnenko_site` в окно загрузки.
4. При желании подключите домен в Domain management.

## Как загрузить на GitHub Pages

1. Создайте публичный репозиторий.
2. Загрузите туда эти файлы.
3. В Settings → Pages выберите ветку `main` и папку `/root`.
4. После публикации подключите домен, если нужен.

## По Google Sites

Google Sites удобен, если вообще не хочется возиться с файлами, но этот сайт написан как обычный HTML/CSS.
Для Google Sites проще пересобрать страницу вручную блоками: заголовок, меню, доставка, контакты.
