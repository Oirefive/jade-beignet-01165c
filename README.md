# Кафе «Вкусненько»

Production-версия статического сайта кафе «Вкусненько» в Феодосии.

## Что внутри

- Главная страница с меню, доставкой, отзывами и контактами.
- Локальная галерея и фото блюд без внешней загрузки изображений.
- Данные меню встроены в `assets/js/data.js`, поэтому сайт стабильно работает и без запроса к JSON.
- Отдельные JSON-файлы в `assets/data/` оставлены для удобного редактирования меню и галереи.
- Конфигурация для Netlify: `_headers`, `netlify.toml`, `404.html`.

## Структура

- `index.html` — главная страница.
- `assets/css/style.css` — стили сайта.
- `assets/js/app.js` — фильтр меню, поиск, галерея и интерактивные элементы.
- `assets/js/data.js` — встроенные данные меню и фотографий.
- `assets/img/photos/` — локальные фотографии.
- `assets/img/fallback/` — резервные SVG-иллюстрации.
- `README_DEPLOY.md` — заметки по публикации и обслуживанию.

## Публикация

Сайт можно разместить на Netlify, Cloudflare Pages, GitHub Pages или обычном статическом хостинге. Для публикации достаточно загрузить содержимое репозитория как статический сайт.

Перед финальным запуском стоит заменить сайт в `robots.txt` и `sitemap.xml` на реальный домен.
